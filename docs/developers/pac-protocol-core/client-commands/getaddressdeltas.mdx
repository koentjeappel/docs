# getaddressdeltas
## Command

```bash
pacprotocol-cli getaddressdeltas
```

Returns all changes for an address (requires addressindex to be enabled).



## Arguments

```json
{
  "addresses"
    [
      "address"  (string) The base58check encoded address
      ,...
    ]
  "start" (number) The start block height
  "end" (number) The end block height
}
```
## Examples
```bash
pacprotocol-cli getaddressdeltas '{"addresses": ["XwnLY9Tf7Zsef8gMGL2fhWA9ZmMjt4KPwg"]}'
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getaddressdeltas", "params": [{"addresses": ["XwnLY9Tf7Zsef8gMGL2fhWA9ZmMjt4KPwg"]}] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```


## Result
```bash
[
  {
    "satoshis"  #(number) The difference of duffs
    "txid"  #(string) The related txid
    "index"  #(number) The related input or output index
    "blockindex"  #(number) The related block index
    "height"  #(number) The block height
    "address"  #(string) The base58check encoded address
  }
]
```
