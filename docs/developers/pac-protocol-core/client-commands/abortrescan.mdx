# abortrescan
## Command

```bash
pacprotocol-cli abortrescan
```

Stops current wallet rescan triggered by an RPC call, e.g. by an importprivkey call.



## Examples


Import a private key

```bash
pacprotocol-cli importprivkey "mykey"
```



Abort the running wallet rescan

```bash
pacprotocol-cli abortrescan 
```



As a JSON-RPC call

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "abortrescan", "params": [] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```
