# listaddressgroupings
## Command

```bash
pacprotocol-cli listaddressgroupings
```

Lists groups of addresses which have had their common ownership
made public by common use as inputs or as the resulting change
in past transactions



## Examples
```bash
pacprotocol-cli listaddressgroupings 
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "listaddressgroupings", "params": [] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```


## Result
```bash
[
  [
    [
      "address",            #(string) The PAC address
      amount,                 #(numeric) The amount in PAC
      "label"               #(string, optional) The label
    ]
    ,...
  ]
  ,...
]
```
