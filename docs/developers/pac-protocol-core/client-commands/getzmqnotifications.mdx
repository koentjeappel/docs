# getzmqnotifications
## Command

```bash
pacprotocol-cli getzmqnotifications
```

Returns information about the active ZeroMQ notifications.



## Examples
```bash
pacprotocol-cli getzmqnotifications 
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getzmqnotifications", "params": [] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```


## Result
```bash
[
  {                        #(json object)
    "type": "pubhashtx",   #(string) Type of notification
    "address": "..."       #(string) Address of the publisher
  },
  ...
]
```

 pacprotocol-cli help getzmqnotifications
