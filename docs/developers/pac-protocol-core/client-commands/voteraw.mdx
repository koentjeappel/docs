# voteraw
## Command

```bash
pacprotocol-cli voteraw <mn-collateral-tx-hash> <mn-collateral-tx-index> <governance-hash> <vote-signal> [yes|no|abstain] <time> <vote-sig>
```





___

Running following help command
```bash
 pacprotocol-cli help voteraw
```
Will produce following output:


```bash
voteraw <mn-collateral-tx-hash> <mn-collateral-tx-index> <governance-hash> <vote-signal> [yes|no|abstain] <time> <vote-sig>
Compile and relay a governance vote with provided external signature instead of signing vote internally

```
