# importwallet
## Command

```bash
pacprotocol-cli importwallet "filename"
```

Imports keys from a wallet dump file (see dumpwallet). Requires a new wallet backup to include imported keys.



## Arguments

### `filename` <span class='asterisk'>*</span>
*(string, required)*

The wallet file


## Examples


Dump the wallet

```bash
pacprotocol-cli dumpwallet "test"
```



Import the wallet

```bash
pacprotocol-cli importwallet "test"
```



Import using the json rpc call

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "importwallet", "params": ["test"] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```
