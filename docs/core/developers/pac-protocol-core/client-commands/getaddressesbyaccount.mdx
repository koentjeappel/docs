# getaddressesbyaccount
## Command

```bash
pacprotocol-cli getaddressesbyaccount "account"
```

DEPRECATED. Returns the list of addresses for the given account.



## Arguments

### `account` <span class='asterisk'>*</span>
*(string, required)*

The account name.


## Examples
```bash
pacglobal-cli getaddressesbyaccount "tabby"
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getaddressesbyaccount", "params": ["tabby"] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```
## Result
```bash
[                     #(json array of string)
  "address"         #(string) a dash address associated with the given account
  ,...
]
```
