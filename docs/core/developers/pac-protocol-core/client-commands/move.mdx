# move
## Command

```bash
pacprotocol-cli move "fromaccount" "toaccount" amount ( minconf "comment" )
```

DEPRECATED. Move a specified amount from one account in your wallet to another.



## Arguments

### `fromaccount` <span class='asterisk'>*</span>
*(string, required)*

The name of the account to move funds from. May be the default account using "".


### `toaccount` <span class='asterisk'>*</span>
*(string, required)*

The name of the account to move funds to. May be the default account using "".


### `amount` 
*(numeric)*

Quantity of PAC to move between accounts.


### `(dummy)` 
*(numeric, optional)*

Ignored. Remains for backward compatibility.


### `comment` 
*(string, optional)*

An optional comment, stored in the wallet only.


## Examples


Move 0.01 PAC from the default account to the account named tabby

```bash
pacglobal-cli move "" "tabby" 0.01
```



Move 0.01 PAC timotei to akiko with a comment and funds have 6 confirmations

```bash
pacglobal-cli move "timotei" "akiko" 0.01 6 "happy birthday!"
```



As a json rpc call

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "move", "params": ["timotei", "akiko", 0.01, 6, "happy birthday!"] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```
## Result
```bash
true|false          #(boolean) true if successful.
```
