# getmerkleblocks
## Command

```bash
pacprotocol-cli getmerkleblocks "filter" "hash" ( count )
```

Returns an array of hex-encoded merkleblocks for &lt;count&gt; blocks starting from &lt;hash&gt; which match &lt;filter&gt;.



## Arguments

### `filter` <span class='asterisk'>*</span>
*(string, required)*

The hex encoded bloom filter


### `hash` <span class='asterisk'>*</span>
*(string, required)*

The block hash


### `count` 
*(numeric, optional, default/max=2000)*




## Examples
```bash
pacglobal-cli getmerkleblocks "2303028005802040100040000008008400048141010000f8400420800080025004000004130000000000000001" "00000000007e1432d2af52e8463278bf556b55cf5049262f25634557e2e91202" 2000
```

## Result
```bash
[
  "data",                        #(string)  A string that is serialized, hex-encoded data for a merkleblock.
  ...
]
```
