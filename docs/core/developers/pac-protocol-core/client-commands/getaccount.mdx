# getaccount
## Command

```bash
pacprotocol-cli getaccount "address"
```

DEPRECATED. Returns the account associated with the given address.



## Arguments

### `address` <span class='asterisk'>*</span>
*(string, required)*

The dash address for account lookup.


## Examples
```bash
pacglobal-cli getaccount "XwnLY9Tf7Zsef8gMGL2fhWA9ZmMjt4KPwG"
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getaccount", "params": ["XwnLY9Tf7Zsef8gMGL2fhWA9ZmMjt4KPwG"] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```
## Result
```bash
"accountname"        #(string) the account address
```
