# setban
## Command

```bash
pacprotocol-cli setban "subnet" "add|remove" (bantime) (absolute)
```

Attempts to add or remove an IP/Subnet from the banned list.



## Arguments

### `subnet` <span class='asterisk'>*</span>
*(string, required)*

The IP/Subnet (see getpeerinfo for nodes IP) with an optional netmask (default is /32 = single IP)


### `command` <span class='asterisk'>*</span>
*(string, required)*

'add' to add an IP/Subnet to the list, 'remove' to remove an IP/Subnet from the list


### `bantime` 
*(numeric, optional)*

time in seconds how long (or until when if  is set) the IP is banned (0 or empty means using the default time of 24h which can also be overwritten by the -bantime startup argument)


```json
[absolute]
```



### `absolute` 
*(boolean, optional)*

If set, the bantime must be an absolute timestamp in seconds since epoch (Jan 1 1970 GMT)


## Examples
```bash
pacglobal-cli setban "192.168.0.6" "add" 86400
```

```bash
pacglobal-cli setban "192.168.0.0/24" "add"
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "setban", "params": ["192.168.0.6", "add", 86400] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```
