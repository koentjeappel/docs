# listaccounts
## Command

```bash
pacprotocol-cli listaccounts ( minconf addlocked include_watchonly)
```

DEPRECATED. Returns Object that has account names as keys, account balances as values.



## Arguments

### `minconf` 
*(numeric, optional, default=1)*

Only include transactions with at least this many confirmations


### `addlocked` 
*(bool, optional, default=false)*

Whether to include transactions locked via InstaPAC.


### `include_watchonly` 
*(bool, optional, default=false)*

Include balances in watch-only addresses (see 'importaddress')


## Examples


List account balances where there at least 1 confirmation

```bash
pacglobal-cli listaccounts 
```



List account balances including zero confirmation transactions

```bash
pacglobal-cli listaccounts 0
```



List account balances for 6 or more confirmations

```bash
pacglobal-cli listaccounts 6
```



As json rpc call

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "listaccounts", "params": [6] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```
## Result
```json
{                    #(json object where keys are account names, and values are numeric balances
  "account": x.xxx,  #(numeric) The property name is the account name, and the value is the total balance for the account.
  ...
}
```
