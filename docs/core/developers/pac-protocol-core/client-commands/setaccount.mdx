# setaccount
## Command

```bash
pacprotocol-cli setaccount "address" "account"
```

DEPRECATED. Sets the account associated with the given address.



## Arguments

### `address` <span class='asterisk'>*</span>
*(string, required)*

The dash address to be associated with an account.


### `account` <span class='asterisk'>*</span>
*(string, required)*

The account to assign the address to.


## Examples
```bash
pacglobal-cli setaccount "XwnLY9Tf7Zsef8gMGL2fhWA9ZmMjt4KPwG" "tabby"
```
