# getaddressutxos
## Command

```bash
pacprotocol-cli getaddressutxos
```

Returns all unspent outputs for an address (requires addressindex to be enabled).



## Arguments

```json
{
  "addresses"
    [
      "address"  (string) The base58check encoded address
      ,...
    ]
}
```
## Examples
```bash
pacglobal-cli getaddressutxos '{"addresses": ["XwnLY9Tf7Zsef8gMGL2fhWA9ZmMjt4KPwg"]}'
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getaddressutxos", "params": [{"addresses": ["XwnLY9Tf7Zsef8gMGL2fhWA9ZmMjt4KPwg"]}] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```


## Result
```bash
[
  {
    "address"  #(string) The address base58check encoded
    "txid"  #(string) The output txid
    "outputIndex"  #(number) The output index
    "script"  #(string) The script hex encoded
    "satoshis"  #(number) The number of duffs of the output
    "height"  #(number) The block height
  }
]
```
