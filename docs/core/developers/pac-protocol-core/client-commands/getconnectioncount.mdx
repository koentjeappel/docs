# getconnectioncount
## Command

```bash
pacprotocol-cli getconnectioncount
```

Returns the number of connections to other nodes.



## Examples
```bash
pacglobal-cli getconnectioncount 
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getconnectioncount", "params": [] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```


## Result
```bash
n          #(numeric) The connection count
```


___

Running following help command
```bash
 pacprotocol-cli help getconnectioncount
```
Will produce following output:


```bash
getconnectioncount
```
