# listwallets
## Command

```bash
pacprotocol-cli listwallets
```

listwallets
Returns a list of currently loaded wallets.
For full information on the wallet, use "getwalletinfo"



## Examples
```bash
pacglobal-cli listwallets 
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "listwallets", "params": [] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```


## Result
```bash
[                         #(json array of strings)
  "walletname"            #(string) the wallet name
   ...
]
```

 pacprotocol-cli help listwallets
