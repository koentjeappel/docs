# setstaking
## Command

```bash
pacprotocol-cli setstaking
```

staking
Returns an object that can be toggled to enable or disable staking.
}



## Examples
```bash
pacglobal-cli setstaking true
```




___

Running following help command
```bash
 pacprotocol-cli help setstaking
```
Will produce following output:


```bash
staking
Returns an object that can be toggled to enable or disable staking.
}

Examples:
> pacglobal-cli setstaking true

```
