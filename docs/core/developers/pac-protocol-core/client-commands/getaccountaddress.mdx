# getaccountaddress
## Command

```bash
pacprotocol-cli getaccountaddress "account"
```

DEPRECATED. Returns the current PACGlobal address for receiving payments to this account.



## Arguments

### `account` <span class='asterisk'>*</span>
*(string, required)*

The account name for the address. It can also be set to the empty string "" to represent the default account. The account does not need to exist, it will be created and a new address created  if there is no account by the given name.


## Examples
```bash
pacglobal-cli getaccountaddress 
```

```bash
pacglobal-cli getaccountaddress ""
```

```bash
pacglobal-cli getaccountaddress "myaccount"
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getaccountaddress", "params": ["myaccount"] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```
## Result
```bash
"address"          #(string) The account dash address
```
