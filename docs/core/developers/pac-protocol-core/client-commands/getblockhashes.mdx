# getblockhashes
## Command

```bash
pacprotocol-cli getblockhashes timestamp
```

Returns array of hashes of blocks within the timestamp range provided.



## Arguments

### `high` <span class='asterisk'>*</span>
*(numeric, required)*

The newer block timestamp


### `low` <span class='asterisk'>*</span>
*(numeric, required)*

The older block timestamp


## Examples
```bash
pacglobal-cli getblockhashes 1231614698 1231024505
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getblockhashes", "params": [1231614698, 1231024505] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```
## Result
```bash
[
  "hash"         #(string) The block hash
]
```
