# importelectrumwallet
## Command

```bash
pacprotocol-cli importelectrumwallet "filename" index
```

Imports keys from an Electrum wallet export file (.csv or .json)



## Arguments

### `filename` <span class='asterisk'>*</span>
*(string, required)*

The Electrum wallet export file, should be in csv or json format


### `index` 
*(numeric, optional, default=0)*

Rescan the wallet for transactions starting from this block index


## Examples


Import the wallet

```bash
pacglobal-cli importelectrumwallet "test.csv"
```

```bash
pacglobal-cli importelectrumwallet "test.json"
```



Import using the json rpc call

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "importelectrumwallet", "params": ["test.csv"] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "importelectrumwallet", "params": ["test.json"] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```
