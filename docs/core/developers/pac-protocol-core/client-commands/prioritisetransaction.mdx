# prioritisetransaction
## Command

```bash
pacprotocol-cli prioritisetransaction <txid> <fee delta>
```

prioritisetransaction &lt;txid&gt; &lt;fee delta&gt;
Accepts the transaction into mined blocks at a higher (or lower) priority



## Arguments

### `txid` <span class='asterisk'>*</span>
*(string, required)*

The transaction id.


### `fee_delta` <span class='asterisk'>*</span>
*(numeric, required)*

The fee value (in duffs) to add (or subtract, if negative).
                  The fee is not actually paid, only the algorithm for selecting transactions into a block
                  considers the transaction as it would have paid a higher (or lower) fee.


## Examples
```bash
pacglobal-cli prioritisetransaction "txid" 10000
```

```bash
true              #(boolean) Returns true
```
