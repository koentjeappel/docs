# setprivatesendrounds
## Command

```bash
pacprotocol-cli setprivatesendrounds rounds
```

Set the number of rounds for PrivateSend mixing.



## Arguments

### `rounds` <span class='asterisk'>*</span>
*(numeric, required)*

The default number of rounds is 4 Cannot be more than 16 nor less than 2



## Examples
```bash
pacglobal-cli setprivatesendrounds 4
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "setprivatesendrounds", "params": [16] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```
