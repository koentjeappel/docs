# getreceivedbyaccount
## Command

```bash
pacprotocol-cli getreceivedbyaccount "account" ( minconf addlocked )
```

DEPRECATED. Returns the total amount received by addresses with &lt;account&gt; in transactions with specified minimum number of confirmations.



## Arguments

### `account` <span class='asterisk'>*</span>
*(string, required)*

The selected account, may be the default account using "".


### `minconf` 
*(numeric, optional, default=1)*

Only include transactions confirmed at least this many times.


### `addlocked` 
*(bool, optional, default=false)*

Whether to include transactions locked via InstaPAC.


## Examples


Amount received by the default account with at least 1 confirmation

```bash
pacglobal-cli getreceivedbyaccount ""
```



Amount received at the tabby account including unconfirmed amounts with zero confirmations

```bash
pacglobal-cli getreceivedbyaccount "tabby" 0
```



The amount with at least 6 confirmations

```bash
pacglobal-cli getreceivedbyaccount "tabby" 6
```



As a json rpc call

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getreceivedbyaccount", "params": ["tabby", 6] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```
## Result
```bash
amount            #(numeric) The total amount in PAC received for this account.
```
