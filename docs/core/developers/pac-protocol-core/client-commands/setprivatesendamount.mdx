# setprivatesendamount
## Command

```bash
pacprotocol-cli setprivatesendamount amount
```

Set the goal amount in PAC for PrivateSend mixing.



## Arguments

### `amount` <span class='asterisk'>*</span>
*(numeric, required)*

The default amount is 100000 Cannot be more than 100000 nor less than 2



## Examples
```bash
pacglobal-cli setprivatesendamount 500
```
