# submitblock
## Command

```bash
pacprotocol-cli submitblock "hexdata"  ( "dummy" )
```

Attempts to submit new block to network.
See https://en.bitcoin.it/wiki/BIP_0022 for full specification.



## Arguments

### `hexdata` <span class='asterisk'>*</span>
*(string, required)*

the hex-encoded block data to submit


### `dummy` 
*(optional)*

dummy value, for compatibility with BIP22. This value is ignored.


## Examples


```bash
pacprotocol-cli submitblock "mydata"
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "submitblock", "params": ["mydata"] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```
