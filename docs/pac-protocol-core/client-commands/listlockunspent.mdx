# listlockunspent
## Command

```bash
pacprotocol-cli listlockunspent
```

Returns list of temporarily unspendable outputs.
See the lockunspent call to lock and unlock transactions for spending.



## Examples


List the unspent transactions

```bash
pacprotocol-cli listunspent 
```



Lock an unspent transaction

```bash
pacprotocol-cli lockunspent false "[{\"txid\":\"a08e6907dbbd3d809776dbfc5d82e371b764ed838b5655e72f463568df1aadf0\",\"vout\":1}]"
```



List the locked transactions

```bash
pacprotocol-cli listlockunspent 
```



Unlock the transaction again

```bash
pacprotocol-cli lockunspent true "[{\"txid\":\"a08e6907dbbd3d809776dbfc5d82e371b764ed838b5655e72f463568df1aadf0\",\"vout\":1}]"
```



As a json rpc call

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "listlockunspent", "params": [] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```


## Result
```bash
[
  {
    "txid" : "transactionid",     #(string) The transaction id locked
    "vout" : n                      #(numeric) The vout value
  }
  ,...
]
```
