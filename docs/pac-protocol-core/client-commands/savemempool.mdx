# savemempool
## Command

```bash
pacprotocol-cli savemempool
```

Dumps the mempool to disk. It will fail until the previous dump is fully loaded.



## Examples
```bash
pacprotocol-cli savemempool 
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "savemempool", "params": [] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```




___

Running following help command
```bash
 pacprotocol-cli help savemempool
```
Will produce following output:


```bash
savemempool
```
