# getaddednodeinfo
## Command

```bash
pacprotocol-cli getaddednodeinfo ( "node" )
```

Returns information about the given added node, or all added nodes
(note that onetry addnodes are not listed here)



## Arguments

### `node` 
*(string, optional)*

If provided, return information about this specific node, otherwise all nodes are returned.


## Examples
```bash
pacprotocol-cli getaddednodeinfo 
```

```bash
pacprotocol-cli getaddednodeinfo "192.168.0.201"
```

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getaddednodeinfo", "params": ["192.168.0.201"] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```
## Result
```bash
[
  {
    "addednode" : "192.168.0.201",   #(string) The node IP address or name #(as provided to addnode)
    "connected" : true|false,          #(boolean) If connected
    "addresses" : [                    #(list of objects) Only when connected = true
       {
         "address" : "192.168.0.201:9999",  #(string) The PAC server IP and port we're connected to
         "connected" : "outbound"           #(string) connection, inbound or outbound
       }
     ]
  }
  ,...
]
```
