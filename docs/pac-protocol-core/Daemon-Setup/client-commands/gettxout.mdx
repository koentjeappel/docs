# gettxout
## Command

```bash
pacprotocol-cli gettxout "txid" n ( include_mempool )
```

Returns details about an unspent transaction output.



## Arguments

### `txid` <span class='asterisk'>*</span>
*(string, required)*

The transaction id


### `n` <span class='asterisk'>*</span>
*(numeric, required)*

vout number


### `include_mempool` 
*(boolean, optional)*

Whether to include the mempool. Default: true.     Note that an unspent output that is spent in the mempool won't appear.


## Examples


Get unspent transactions

```bash
pacprotocol-cli listunspent 
```



View the details

```bash
pacprotocol-cli gettxout "txid" 1
```



As a json rpc call

```bash
curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "gettxout", "params": ["txid", 1] }' -H 'content-type: text/plain;' http://127.0.0.1:1111/
```
## Result
```json
{
  "bestblock":  "hash",    #(string) The hash of the block at the tip of the chain
  "confirmations" : n,       #(numeric) The number of confirmations
  "value" : x.xxx,           #(numeric) The transaction value in PAC
  "scriptPubKey" : {         #(json object)
     "asm" : "code",       #(string) 
     "hex" : "hex",        #(string) 
     "reqSigs" : n,          #(numeric) Number of required signatures
     "type" : "pubkeyhash", #(string) The type, eg pubkeyhash
     "addresses" : [          #(array of string) array of PAC addresses
        "address"     #(string) PAC address
        ,...
     ]
  },
  "coinbase" : true|false   #(boolean) Coinbase or not
}
```
