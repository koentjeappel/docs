PAC Protocol is an open source network developed by PAC Global Holdings LLC. 
PAC Protocol is built on the foundation of a first-of-its-kind blockchain technology utilizing both proof-of-stake (POSv3, economically friendly) and deterministic Masternodes to offer a large globally decentralized network, 
proven to boast scalability with over 22,000 Masternodes worldwide and growing. This creates the digital architecture required for decentralized data storage, content management, and more using IPFS software.
